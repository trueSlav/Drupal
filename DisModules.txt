Actions
Позволяет выполнять настройку задач в ответ на события.

Activity Tracker	
Включает отслеживание последнего содержимого для пользователей.

Aggregator (Deprecated)	
Собирает и отображает синдицированный контент (RSS, RDF и каналы Atom) из внешних источников.

Automated Cron	
Обеспечивает автоматизированный способ запуска задач планировщика cron, выполняя их по завершении ответов сервера.

Ban	
Позволяет администраторам запрещать посещения с определенных IP-адресов.

BigPipe	
Передаёт страницы с помощью технологии BigPipe, которая позволяет браузерам показывать их значительно быстрее.

Block	
Управление визуальным построением блоков, из которых состоит страница. Блоки — это прямоугольники с содержимым, которые отображаются в области веб-страницы.

Book	
Позволяет пользователям создавать и организовывать связанные материалы в оглавление.

Breakpoint	
Управляет точками останова и группами точек останова для адаптивных дизайнов.

CKEditor	
WYSIWYG-редактирование для форматированного текста с использованием CKEditor.

Color (Deprecated)	
Позволяет пользователям изменять цветовую схему совместимых тем.

Comment	
Allows users to comment on content.

Configuration Manager	
Позволяет администраторам импортировать и экспортировать изменения конфигурации.

Contact	
Предоставляет общедоступные контактные формы и формы для связи с отдельными пользователями.

Content Moderation	
Предоставление статуса модерации для контента.

Contextual Links	
Предоставляет контекстные ссылки для прямого доступа к задачам, связанным с элементами страницы.

Custom Block	
Позволяет создавать пользовательские блоки и типы блоков.

Custom Menu Links	
Позволять пользователям создавать ссылки меню.

Database Logging	
Записывает системные события в базе данных.

Field	
Field API для добавления полей различным объектам.

Field UI	
Предоставляет пользовательский интерфейс для модуля Field.

Filter	
Фильтры содержимого в рамках подготовки к отображению.

Forum	
Ведение обсуждений.

Help	
Управляет показом онлайновой справки.

History	
Записи о том какой пользователь какое содержимое просматривал.

Inline Form Errors	
Размещает сообщения об ошибках рядом с полями ввода формы для повышения удобства использования и доступности.

Internal Dynamic Page Cache	
Кэширует страницы, в том числе с динамическим контентом, для всех пользователей.

Internal Page Cache	
Кэширует страницы для анонимных пользователей и может использоваться, когда внешний кеш страниц недоступен.

Layout Builder	
Позволяет пользователям добавлять и упорядочивать блоки и поля контента непосредственно в контенте.

Layout Discovery	
Предоставляет возможность модулям или темам регистрировать макеты.

Media	
Управляет созданием, настройкой и отображением элементов медиа.

Media Library	
Расширяет список медиа с помощью дополнительных функций для более удобного поиска и использования существующих элементов медиа.

Menu UI	
Позволяет администраторам настраивать меню навигации сайта.

MySQL	
Драйвер базы данных MySQL.

Node	
Позволяет добавлять материалы и отображать их на страницах сайта.

Path	
Позволяет пользователям переименовывать URL-адреса.

PostgreSQL	
БД драйвер для PostgreSQL.

Quick Edit	
Быстрое редактирование материала.

RDF	
Дополняет материал метаданными, что позволяет другим приложениям (таким, как поисковые системы, сборщики лент) лучше распознавать связи и атрибуты материала.

Responsive Image	
Предоставляет средства форматирования изображения и сопоставления контрольных точек для вывода адаптивных изображений с помощью тега изображения HTML5.

Search	
Позволяет пользователям создавать поисковые страницы на основе плагинов предоставленных другими модулями.

Settings Tray	
Позволяет пользователям напрямую редактировать конфигурацию блоков на текущей странице.

Shortcut

SQLite

System

Taxonomy

Text Editor	
Предоставляет средства для связи текстовых форматов с библиотеками текстового редактора, такими как WYSIWYG или панели инструментов.

Toolbar	
Предоставляет панель инструментов, которая показывает ссылки меню администрирования верхнего уровня и ссылки из других модулей.

Tour

User

Views

Workflows
